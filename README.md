メモ ギークな感じのアプリ
- NFC Tools
- CloudBeats
  - Nextcloud/Owncloud上のファイルを再生できる音楽プレーヤーアプリ
- Button Mapper
  - Android端末の物理ボタンの再マッピングができる。Xperiaのシャッターボタン全押し&長押し→PhotoPro、半押し&長押し→VideoPro みたいな使い分けができる
- CameraFi Live
  - スマホカメラor画面の映像をストリーミングできる。RTSP/SRT/RTMP対応、OBSに取り込みもできる
- WiFi Analyzer
  - そこらへんで飛んでるWiFiの電波を可視化できる。めっちゃ干渉してるな～とかわかる
- Network Cell Info
  - つないでる基地局の情報を見れる
- Device Info HW
  - ハードウェア情報を見れる
- Precise Volume
  - 音量を細かく調整できるようになる。最小音量なのにイヤホンの音がでかすぎるときにも使える
